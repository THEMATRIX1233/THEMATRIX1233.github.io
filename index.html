      01 — Historia
      
Nacido en 10 días

      
La historia detrás del lenguaje más influyente de la web moderna.

    

    

      

        

        

          1995
          
El Nacimiento

          
Brendan Eich crea JavaScript en solo 10 días mientras trabaja en Netscape. Originalmente llamado "Mocha", luego "LiveScript" y finalmente JavaScript por motivos de marketing.

        

      

      

        

        

          1997
          
ECMAScript 1

          
ECMA International estandariza JavaScript como ECMAScript. Se establece la base del lenguaje que conocemos hoy.

        

      

      

        

        

          2005
          
La Era Ajax

          
Jesse James Garrett acuña el término Ajax. Google Maps y Gmail demuestran el poder de las aplicaciones web dinámicas. JS se convierte en algo serio.

        

      

      

        

        

          2009
          
Node.js & ES5

          
Ryan Dahl libera Node.js, llevando JavaScript al servidor. ES5 añade JSON nativo, getters/setters y modo estricto.

        

      

      

        

        

          2015
          
ES6 — La Revolución

          
ECMAScript 2015 transforma el lenguaje: arrow functions, clases, módulos, Promises, destructuring, template literals y mucho más.

        

      

      

        

        

          Hoy
          
El Lenguaje Universal

          
JavaScript domina frontend, backend, apps móviles (React Native), desktop (Electron), IoT y hasta machine learning con TensorFlow.js.

        

      

    

  


  
  

    

      02 — Fundamentos
      
Los Pilares del Lenguaje

      
Domina los conceptos core que todo desarrollador JS debe conocer.

    

    

      

        
📦

        
Variables & Tipos

        
JavaScript es dinámicamente tipado. Usa let, const y var para declarar variables. Los tipos primitivos son: string, number, boolean, null, undefined, symbol y bigint.

        
Primitivos

      

      

        
⚙️

        
Funciones

        
Ciudadanos de primera clase. Pueden ser declaradas, expresadas o arrow functions. Soportan closures, callbacks, IIFE y son la base de la programación funcional en JS.

        
First-class

      

      

        
🗂️

        
Objetos & Arrays

        
Todo en JS es (casi) un objeto. Los arrays son objetos con índices numéricos. El prototype chain permite la herencia. La destructuring simplifica el acceso a propiedades.

        
Referencia

      

      

        
🔄

        
Event Loop

        
JS es single-threaded pero no-bloqueante. El Event Loop, la Call Stack, el Heap y las Callback/Microtask Queues trabajan juntos para manejar operaciones asíncronas.

        
Async

      

      

        
🔒

        
Scope & Closure

        
El scope define dónde son accesibles las variables. Los closures permiten que las funciones recuerden su entorno léxico, creando datos privados y funciones de fábrica.

        
Léxico

      

      

        
🧬

        
Prototype Chain

        
JavaScript usa herencia prototípica, no clásica. Cada objeto tiene un [[Prototype]] interno. Las clases de ES6 son syntactic sugar sobre este mecanismo.

        
OOP

      

      

        
🌐

        
DOM & BOM

        
El Document Object Model permite manipular HTML/CSS. El Browser Object Model da acceso a la ventana, historial, geolocalización y más capacidades del navegador.

        
Web APIs

      

      

        
❌

        
Manejo de Errores

        
Usa try/catch/finally para errores síncronos. .catch() y async/await con try/catch para promesas. Los errores personalizados extienden Error.

        
Robustez

      

    

  


  
  

    

      03 — Avanzado
      
ES6+ y Más Allá

      
Las características modernas que hacen a JS un lenguaje poderoso y expresivo.

    

    

      

        
        

          
Las Promises representan un valor futuro. async/await es syntactic sugar que hace el código asíncrono parecer síncrono. El método Promise.all() ejecuta promesas en paralelo.

          
async function fetchData() {
  try {
    const res = await fetch('/api/data');
    const data = await res.json();
    return data;
  } catch (err) {
    console.error(err);
  }
}

        

      

      

        
        

          
Destructuring permite extraer valores de arrays y objetos de forma concisa. El operador spread (...) expande iterables y el rest agrupa el resto.

          
const { nombre, edad, ...resto } = usuario;
const [primero, segundo, ...otros] = lista;
const nuevo = { ...obj1, ...obj2, extra: true };

        

      

      

        
        

          
Los métodos de array de orden superior son la columna vertebral de la programación funcional en JS. map transforma, filter filtra y reduce acumula.

          
const resultado = datos
  .filter(x => x.activo)
  .map(x => ({ ...x, score: x.valor * 2 }))
  .reduce((acc, x) => acc + x.score, 0);

        

      

      

        
        

          
Los generadores (function*) producen valores bajo demanda con yield. Los iteradores implementan el protocolo [Symbol.iterator] para objetos iterables personalizados.

          
function* infiniteCounter() {
  let n = 0;
  while (true) yield n++;
}
const gen = infiniteCounter();
console.log(gen.next().value); // 0, 1, 2...

        

      

      

        
        

          
Proxy intercepta operaciones sobre objetos (get, set, delete). Reflect provee métodos para las operaciones básicas de JS. Son la base de frameworks de reactividad como Vue 3.

          
const handler = {
  get(obj, key) { return key in obj ? obj[key] : 37; }
};
const proxy = new Proxy({}, handler);

        

      

      

        
        

          
Los módulos ES6 permiten dividir el código en archivos reutilizables con import/export. Los imports son estáticos (tree-shakeable) y pueden ser dinámicos con import().

          
// math.js
export const sum = (a, b) => a + b;
// main.js
import { sum } from './math.js';
const chunk = await import('./lazy.js');

        

      

    

  


  
  

    

      04 — Ecosistema
      
El Universo JS

      
Frameworks, runtimes y herramientas que conforman el ecosistema más grande del mundo.

    

    

      

        
        
        
        
      

      

        

          

            
⚛

            
React

            
Librería de UI de Meta. Virtual DOM, hooks, JSX. El más popular del mundo. Impulsado por Next.js.

            

          

          

            
V

            
Vue.js

            
Framework progresivo. Sistema de reactividad con Proxy, Composition API, excelente DX y curva suave.

            

          

          

            
A

            
Angular

            
Framework completo de Google. TypeScript-first, inyección de dependencias, ideal para enterprise.

            

          

          

            
S

            
Svelte

            
Compila a JS puro sin virtual DOM. Código más conciso, bundle más pequeño y excelente rendimiento.

            

          

        

      

      

        

          

            
⬡

            
Node.js

            
Runtime de JS en el servidor. V8 engine, event-driven, non-blocking I/O. El estándar de la industria.

            

          

          

            
🦕

            
Deno

            
Runtime seguro por defecto de Ryan Dahl. TypeScript nativo, permisos granulares, Web APIs estándar.

            

          

          

            
🍞

            
Bun

            
Runtime ultra-rápido escrito en Zig. Incluye bundler, test runner y gestor de paquetes integrados.

            

          

          

            
Ex

            
Express / Fastify

            
Frameworks web minimalistas para Node.js. Express es el veterano; Fastify prioriza el rendimiento.

            

          

        

      

      

        

          

            
⚡

            
Vite

            
Build tool ultra-rápido basado en ESM nativo. HMR instantáneo, optimización para producción con Rollup.

            

          

          

            
TS

            
TypeScript

            
Superset tipado de JS. Sistema de tipos estático, inferencia, genéricos. El estándar de proyectos serios.

            

          

          

            
npm

            
npm / pnpm

            
Gestores de paquetes. El registro npm tiene +2M paquetes. pnpm es más rápido y eficiente en disco.

            

          

          

            
ES

            
ESLint + Prettier

            
ESLint analiza y corrige problemas de código. Prettier formatea automáticamente. Indispensables en equipos.

            

          

        

      

      

        

          

            
🃏

            
Jest

            
Framework de testing de Meta. Snapshots, mocking, coverage integrado. El más popular para unit testing.

            

          

          

            
Vi

            
Vitest

            
Testing nativo para Vite. Compatible con la API de Jest. Extremadamente rápido con HMR de tests.

            

          

          

            
Cy

            
Cypress

            
E2E testing moderno. Runs en el navegador, time-travel debugging, screenshots y video automáticos.

            

          

          

            
Pw

            
Playwright

            
E2E de Microsoft. Soporta Chromium, Firefox y WebKit. Auto-wait, trace viewer y test generator.

      06 — Quirks
      
Las Curiosidades de JS

      
Las rarezas que hacen a JavaScript... interesante.

    

    

      

        
typeof null

        
→ "object"

        
Un bug histórico de JS. null no es un objeto, pero typeof lo reporta así desde 1995.

      

      

        
0.1 + 0.2

        
→ 0.30000000000000004

        
Aritmética de punto flotante IEEE 754. No es un bug de JS, sino de cómo las computadoras manejan decimales.

      

      

        
[] + []

        
→ ""

        
Coerción de tipos en acción. Los arrays se convierten en strings vacíos y se concatenan.

      

      

        
NaN === NaN

        
→ false

        
NaN (Not a Number) es el único valor en JS que no es igual a sí mismo. Usa Number.isNaN() para verificar.

      

      

        
typeof undefined

        
→ "undefined"

        
undefined es un tipo primitivo. Se puede usar como variable en contextos no estrictos (¡no lo hagas!).

      

      

        
"5" - 3

        
→ 2

        
El operador - fuerza conversión numérica. Pero "5" + 3 da "53" porque + también concatena strings.
